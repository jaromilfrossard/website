permuco is an R package that provide user and simple way to compute permutation for for ANOVA, regression parameters and repeated measures ANOVA. An extension for massive univariate tests that implements cluster-mass statistics, the threshold-free cluster-enhancement and several other multiple comparison procedure allows to users to computed intuitively tests of difference of amplitude on signals.
