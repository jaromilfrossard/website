Using the R package rayrender, it is really easy to quickly produce 3D images and videos. Here you will learn how to produce a vizualisation of a full-scalp cluster-mass test of EEG data using rayrender. The EEG data come from Cheval et al. (2018) which you can already repoduce using the clustergraph and following a previous post. In the clustergraph package, the build-in image() function allows produce visualistion of the results of the test and, here, we propose an alternative version of this visualisation.

Before following the next steps make sure you have:

  1. Install rayrender and read the introduction tutorial.
  2. Install the clustergraph package. Run the script in my previous post in order to:
      1. Download the electrode position data from <www.biosemi.com> and save it as “Cap_coords_all.xls”
      1. Save the “model” object from the clustergraph() function in file named “model.RData”
  3. Downlaod a 3D model of female-head here: https://free3d.com/3d-model/femalehead-v4--971578.html
